'text-indent' - wciecie tekstu


Wartość: | | inherit
Domyślnie: 0
Dotyczy: elementów blokowych
Np:
P { text-indent: 3em }


'text-align'


Wartość: left | right | center | justify | | inherit
Domyślnie: depends on user agent and writing direction
Dotyczy: elementów blokowych
Np:

DIV.center { text-align: center }

'text-decoration'


Wartość: none | [ underline || overline || line-through || blink ] | inherit
Domyślnie: none
Dotyczy: wszystkich elementów
Np:

A[href] { text-decoration: underline }

'text-shadow'


Wartość: none | [ || ? ,]* [ || ?] | inherit
Domyślnie: none
Dotyczy: wszystkich elementów

Text shadow moze bys użyty z :first-letter i :first-line pseudo-selektorami.

H2 { text-shadow: 3px 3px red, yellow -3px 3px 2px, 3px -3px }


'letter-spacing'


Wartość: normal | | inherit
Domyślnie: normal
Dotyczy: wszystkich elementów
Np:
BLOCKQUOTE { letter-spacing: 0.1em }

'text-transform'


Wartość: capitalize | uppercase | lowercase | none | inherit
Domyślnie: none
Dotyczy: wszystkich elementów
Np:
H1 { text-transform: uppercase }

'white-space'


Wartość: normal | pre | nowrap | inherit
Domyślnie: normal
Dotyczy: elementów blokowych
Np:

PRE { white-space: pre }
P { white-space: normal }
