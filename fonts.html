'font-family'


Wartość: [[ | ],]* [ | ] | inherit
Domyślnie: depends on user agent
Dotyczy: Wszystkich elementów
Np:

BODY { font-family: "Verdana", "Symbol", "serif" }

'font-style'


Wartość: normal | italic | oblique | inherit
Domyślnie: normal
Dotyczy: Wszytskich elementów
Np:
Np: H1, H2, H3 { font-style: italic }


'font-variant'


Wartość: normal | small-caps | inherit
Domyślnie: normal
Dotyczy: Wszytskich elementów
Np
H3 { font-variant: small-caps }

'font-weight'


Wartość: normal | bold | bolder | lighter | 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900 | inherit
Domyślnie: normal
Dotyczy: Wszytskich elementów
Np:

P { font-weight: normal } /* 400 */
H1 { font-weight: 700 } /* bold */
BODY { font-weight: 400 }

'font-stretch'


Wartość: normal | wider | narrower | ultra-condensed | extra-condensed | condensed | semi-condensed | semi-expanded | expanded | extra-expanded | ultra-expanded | inherit
Domyślnie: normal
Dotyczy: Wszystkich elementów

'font-size'


Wartość: | | | | inherit
Domyślnie: medium
Dotyczy: Wszystkich elementów



A Possible Wartośćs are:
[ xx-small | x-small | small | medium | large | x-large | xx-large ]

mozliwe Wartości are: [ larger | smaller ]

Np:
P { font-size: 12pt; }
BLOCKQUOTE { font-size: larger }
EM { font-size: 150% }
EM { font-size: 1.5em }

'font-size-adjust'


Wartość: | none | inherit
Domyślnie: none
Dotyczy: Wszytskich elementów



'font'


Wartość: [ [ <'font-style'> || <'font-variant'> || <'font-weight'> ]? <'font-size'> [ / <'line-height'> ]?
<'font-family'> ] | caption | icon | menu | message-box | small-caption | status-bar | inherit
Domyślnie: see individual properties
Dotyczy: Wszystkich elementów
Np:

P { font: 12pt/14pt sans-serif }
P { font: 80% sans-serif }
P { font: x-large/110% "new century schoolbook", serif }
P { font: bold italic large Palatino, serif }
P { font: normal small-caps 120%/120% fantasy }
P { font: oblique 12pt "Helvetica Nue", serif; font-stretch: condensed }

Zalecane wartości dla zbioru czcionek:
caption
dla sterowania (e.g., buttons, drop-downs, etc.).
icon
dla etykiet ikon
menu
dla menu (e.g., dropdown menus and menu lists).
message-box
dla dialogowych boxów.
small-caption
dla etykiet small controls.
status-bar
dla window status bars.
