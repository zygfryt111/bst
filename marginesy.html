Pudełka(Box) - wymiary



Każdy box ma swoja zawartośc (content) tj (e.g., text, image, etc.) i opcjonalne obszary dla
dopełnienia( padding), obramowaniem (border), marginesami(margin) wymiary każdego z obszarów są specyfikowane
we własnościach definiowanych ja niżej. Ponizszy diagram pokazuje zależnosci i terminologię używaną
dla własnosci : margin, border, and padding:






Rysunek przedstawia relacje pomiedzy zawartością (content), dopełnienm (padding),
obramoaniem(borders), i marginesami( margins).


'margin-top', 'margin-right', 'margin-bottom', 'margin-left'


Wartość: < margin-width > | inherit
Domyślnie: 0
Dotyczy: wszyskich elementów
Np.:
H1 { margin-top: 2em }



margin'


Wartość: {1,4} | inherit
Domyślnie: zaleznie od indywidualnych własności
Dotyczy: wszyskich elementów
Uwaga: Własnośc 'margin' jest skrócona wersją dla własności
'margin-top', 'margin-right', 'margin-bottom', i 'margin-left'.

Np:

BODY { margin: 2em } /* all margins set to 2em */
BODY { margin: 1em 2em } /* top & bottom = 1em, right & left = 2em */
BODY { margin: 1em 2em 3em } /* top=1em, right=2em, bottom=3em, left=2em */

Ostania reguła jest równoważna poniższym 4-rem regułom
BODY {
margin-top: 1em;
margin-right: 2em;
margin-bottom: 3em;
margin-left: 2em;
}



'padding-top', 'padding-right', 'padding-bottom', 'padding-left'


Wartość: | inherit
Domyślnie: 0
Dotyczy: wszyskich elementów
np:
BLOCKQUOTE { padding-top: 0.3em }





'padding'


Wartość: {1,4} | inherit
Domyślnie: nie są okreslone
Dotyczy: wszyskich elementów
Uwaga: Własnośc 'padding' jest skrócona wersją ustawiana własności
'padding-top', 'padding-right', 'padding-bottom', and 'padding-left'

Np:
H1 {
background: white;
padding: 1em 2em;
}




'border-top-width', 'border-right-width', 'border-bottom-width', 'border-left-width'


Wartość: | inherit
Domyślnie: medium
Dotyczy: wszyskich elementów



'border-width'


Wartość: {1,4} | inherit
Domyślnie: zaleznie od indywidualnych własności
Dotyczy: wszyskich elementów
Uwaga:Własnośc 'border-width' jest skrócona wersją ustawiana własności
'border-top-width', 'border-right-width', 'border-bottom-width', and 'border-left-width'

Np:
H1 { border-width: thin } /* thin thin thin thin */
H1 { border-width: thin thick } /* thin thick thin thick */
H1 { border-width: thin thick medium } /* thin thick medium thick */



'border-top-color', 'border-right-color', 'border-bottom-color', 'border-left-color'


Wartość: | inherit
Domyślnie: Wartość własnosci 'color'
Dotyczy: wszyskich elementów



'border-color'


Wartość: {1,4} | transparent | inherit
Domyślnie: zaleznie od indywidualnych własności
Dotyczy: wszyskich elementów




'border-top-style', 'border-right-style', 'border-bottom-style', 'border-left-style'


Wartość: | inherit
Domyślnie: none
Dotyczy: wszyskich elementów



'border-style'


Wartość: {1,4} | inherit
Domyślnie: zaleznie od indywidualnych własności
Dotyczy: wszyskich elementów

#xy34 { border-style: solid dotted }

W przykładzie powyżej pozioma linie obramowania sa ciągłe a pionowe są kropkowane.



'border-top', 'border-right', 'border-bottom', 'border-left'


Wartość: [ <'border-top-width'> || <'border-style'> || ] | inherit
Domyślnie: zaleznie od indywidualnych własności
Dotyczy: wszyskich elementów

H1 { border-bottom: thick solid red }

The above rule will set the width, style, and color of the border below the H1 element. Omitted Wartośćs are set to their Domyślnie Wartośćs. Since the following rule does not specify a border color, the border will have the color specified by the 'color' property:

H1 { border-bottom: thick solid }



'border'


Wartość: [ <'border-width'> || <'border-style'> || ] | inherit
Domyślnie: zaleznie od indywidualnych własności
Dotyczy: wszyskich elementów
Uwaga: własnośc 'border' jest skrócona wersją dla własności
width, color, i style dla obramowania boxów.


Na przykład ponizsze dwie regułu sa równowazne

P { border: solid red }
P {
border-top: solid red;
border-right: solid red;
border-bottom: solid red;
border-left: solid red

}
