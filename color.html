'color'


Value: | inherit
Domyslnie: zaleznie od indywidualnych własności
Dotyczy: Wszystkich elemntów
Np:

EM { color: red } /* predefined color name */
EM { color: rgb(255,0,0) } /* RGB range 0-255 */


'background-color'


Wartość: | transparent | inherit
Domyslnie: transparent
Dotyczy: Wszystkich elemntów
Np:
H1 { background-color: #F00 }

'background-image'


Wartość: | none | inherit
Domyslnie: none
Dotyczy: Wszystkich elemntów
Np:
BODY { background-image: url("marble.gif") }
P { background-image: none }


'background-repeat'


Wartość: repeat | repeat-x | repeat-y | no-repeat | inherit
Domyslnie: repeat
Dotyczy: Wszystkich elemntów

repeat
powtarzanie image poziomo i pionowo .
repeat-x
powtarzanie image tylko poziomo .
repeat-y
powtarzanie image tylko pionowo.
no-repeat
bez powtarzania

Np:

BODY {
background: white url("pendant.gif");
background-repeat: repeat-y;
background-position: center;
}


'background-attachment'


Wartość: scroll | fixed | inherit
Domyslnie: scroll
Dotyczy: Wszystkich elemntów
Np:


BODY {
background: red url("pendant.gif");
background-repeat: repeat-y;
background-attachment: fixed;

'background-position'


Wartość: [ [ | ]{1,2} | [ [top | center | bottom] || [left | center | right] ] ] | inherit
Domyslnie: 0% 0%
Np:

BODY {
background-image: url("logo.png");
background-attachment: fixed;
background-position: 100% 100%;
background-repeat: no-repeat;
}


'background'


Wartość: [<'background-color'> || <'background-image'> || <'background-repeat'> || <'background-attachment'> || <'background-position'>] | inherit
Domyslnie: nie definowane
Dotyczy: Wszystkich elemntów

P { background: url("chess.png") gray 50% repeat fixed }
